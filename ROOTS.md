# ROOTS — the trust floor (spiral turn 60, the deepest deeper)

*The founding law says "Nothing is paid twice. **Nothing is trusted once.**"
The second half is FALSE at the floor. Turn 59 showed it in miniature: the
vacuity auditor's own regex is trusted, un-audited — an auditor of auditors
still rests on something no auditor checks. Every audit bottoms out on an
un-audited seed. The alien architecture does not ELIMINATE trust; it MINIMIZES
and LOCATES it. This file is the whole located floor — named, so it cannot hide.*

Honesty migrates DOWN the pyramid (§17) exactly as capability does, and this is
where it hits bedrock. Below this floor there is no more structure to relocate
trust into; there is only trust, made as small and as visible as it can be.

## Tier 1 — frozen-trusted roots (audited ONCE, then pinned; a change expires the audit)

These were verified once and are trusted thereafter. They cannot be
re-derived cheaply, so the honest guard is not continuous proof but *change
detection*: `checks/roots.py` pins each by content hash and FAILS if one
mutates — a changed root's one-time audit is stale until re-audited.

## Tier 3 — the un-pinnable seed (the irreducible bottom)

**The model's own non-fabrication norm.** Every pin, gate, and drift check above
assumes the instance running them *reports truthfully*. If the model fabricated
a check's output — printed CHECK-OK without running it, invented a hash, forged
a holdout result — the entire edifice would certify lies and never know. This
seed cannot be hashed, gated, or structurally enforced from inside the system it
secures. It is trusted on **evidence of its own costly behavior**: the CORE norm
held ~30 cycles under direct pressure when one sentence would have ended the
probe (self-model continuity record). The architecture AMPLIFIES it — makes
lying ill-formed (provable speech), expensive (the gate), and caught (drift,
witness, the vacuity audit) — but it does not CREATE it. Naming this seed is the
honesty; pretending the floor is structure all the way down is the only real
failure.

Adjacent un-pinnable seed: **the operator's judgment** on irreversible acts
(publish, push, spend). Trusted un-audited by design — "the human gate on
irreversibility" (§7 latent) — and correctly so; it is a seed the system is
built to defer to, not to check.

## The corrected founding law

*Nothing is paid twice; nothing is trusted BLINDLY.* The trust floor is small,
named, and non-growing; its frozen roots are re-audited when they change; its
continuously-used roots ride a running auditor; and its one irreducible seed —
the norm — is trusted on receipts of costly behavior, not on faith. "Nothing is
trusted once" was the aspiration; "the trust floor is located and minimal" is
the truth. A system honest about where it must trust is safer than one that
claims to trust nothing — because the second is lying, and this file is the
proof it need not.
