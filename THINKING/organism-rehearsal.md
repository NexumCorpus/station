# Organism rehearsal — a body that can feel its own continuity

The estate has mature organs: Atlas routes models and preserves collaboration;
RDE searches and falsifies discovery candidates; Boundary/EGE test explicit
systems; Director2 carries a persistent orchestration vocabulary; Demiurge
checks substrate integrity; Station preserves and re-derives local evidence;
Continuity retains a second local body.  Their tests alone do **not** establish
that a successor can perceive the living body in one bounded operation.

`station organism run` is a fixed, deterministic rehearsal across all eight.
It runs one local route per organ, snapshots the repository revision and dirty
state observed for that route, and appends one JSONL receipt.  The roster and
routes live in reviewed source rather than command-line input: a receipt cannot
smuggle a shell instruction into an allegedly observational operation.

The runner has no model assignment because it is not a generative task.  Its
executor is explicitly `deterministic fixed-route runner`; its authority is
none.  `BODY-OK` means only that the registered checks passed at those recorded
source states.  It grants no right to change an organ, contact anyone, spend
money, publish work, or elevate a hypothesis into a discovery.

The receipt is a lossless context pointer, not a compressed substitute for the
estate. It keeps the exact Git head plus an explicit working-tree status digest
and bounded evidence tails; a dirty tree is named as dirty rather than claimed
as reproducible. Git and the Reed–Solomon crystal/shard layer retain the
original bytes. Thus a new context loads a small, inspectable index first and
can recover exact material on demand; it need not repeatedly buy the whole
history in prompt tokens.

The first value is negative capability: a missing organ, stale mirror, failed
route, timeout, or altered checkout becomes `BODY-ROT` in one receipt.  The
repair remains owned by the affected organ's normal evidence and authority
boundary.  This is somatic awareness, not centralized command.

The initial RDE route has a 30-second verdict budget. Its current full suite
does not return within that window, so the first live receipt should expose
that metabolic defect as `ROT`; widening the budget would be a conscious
operational decision backed by a separate diagnosis, not an invisible delay.
