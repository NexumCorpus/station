# Counterfactual immunity — the organism must feel a wound

## The missing causal test

An executable check that passes on today's body establishes only a correlation:
the body and the verdict coexist. It does not establish that the checker would
notice the loss of the property it is meant to guard. A green check can be a
decoration, a vacuous universal, or a test that never reaches the changed seam.

The counterfactual-immunity organ makes one smaller claim. A pre-registered,
single-site lesion is applied only to a disposable copy of the registered suite
root. The unmodified copy must pass its suite; the wounded copy must fail the
same suite. Only that pair earns `KILLED` for that exact lesion.

```
live body ──baseline suite──> PASS
     │
     └─copy + one declared lesion──same suite──> FAIL = KILLED
```

The specimen never touches the live tree. The suite is selected from
`station.json`, not supplied by the trial, so a trial cannot nominate a friendly
checker. The search text must occur exactly once, and the original source hash
travels with the outcome. A later source edit makes the result `STALE`; replay,
not rhetoric, restores it.

## What this does and does not mean

`KILLED` means the named suite discriminated one concrete failure mode. It does
not prove complete test coverage, safety, a general causal model, or that an
unseen mutation will fail. `SURVIVED` is a successful immune finding: it means a
supposed guard failed to feel the wound and should not be counted as protection.
`BASELINE-BROKEN` is no result at all.

This is a recursion primitive. A future improver may propose a change, but its
retained tests are themselves subjected to counterfactual injury. The organism
therefore improves not merely by accumulating checks, but by selecting checks
that can distinguish a world with their guarded property from a nearby world
without it.

## Initial boundary

Only registered local suites are eligible. The runner copies one suite root to a
temporary directory, makes no network call, and retains no arbitrary shell
command in a trial. Initial lesions are intentionally small and textual; more
powerful mutators belong behind this organ only after they can preserve the same
isolation and replay contract.
